mac环境下使用github + hexo 搭建博客

hexo官网

需要环境

  1. git
  2. github userNmae
  3. node.js
  4. 安装hexo

如果是开发的话1.2则是开发小哥必需品吧，则不多介绍,移步官网

www.github.com

https://git-scm.com/

关于node.js,用来生成静态页面的
到Node.js官网下载相应平台的最新版本，一路安装即可;

着重说说这个hexo的坑

关联github

在安装hexo之前一定要确保满足以上3个条件，才能进行下一步

  • 执行如下命令安装Hexo：sudo npm install -g hexo
  • 安装成功以后 hexo init blog(即blog目录)
  • blog目录／source／_posts/则会看见hello world.md 文档了
  • hexo g && hexo d 发布 且push到github
