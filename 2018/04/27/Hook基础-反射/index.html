Hook基础-反射

通过Class.forName静态方法可以根据类的绝对路径拿到某个类的Class实例

新建测试Test类如下

 package com.example.lib;

public class Test {

    private String username;
    private String password;

    public Test(){
    }

    public int showInfo(String name,String pwd){
        MyClass.print("name = " + name);
        MyClass.print("pwd = " + pwd);
        return 2;
    }

    @Override
    public String toString() {
        return "Test{" +
                "username='" + username + '\'' +
                ", password='" + password + '\'' +
                '}';
    }
}

反射变量

public static void main(String[] args){
    Class<?> cls = Class.forName("x.com.lib.Test");
        Object object = cls.newInstance();
        Field username = cls.getDeclaredField("username");
        username.setAccessible(true);
        username.set(object,"100");
}

输出
Test{username='100', password='null'}

注意

  • set(Object var1, Object var2) 方法接收两个参数

    var1 : 当前实例
    var2 : 修改的实例

  • get(Object var1) 方法接受一个参数

    var1 : Field实例

  • 如果没加username.setAccessible(true)则会报错如下

    can not access a member of class x.com.lib.Test with modifiers “private”

因为Test里面定义的变量是private的,改成public即可。

如果需要获取到private属性的则需要加username.setAccessible(true)。

public static void main(String[] args){
    Class<?> cls = Class.forName("x.com.lib.Test");
            Object object = cls.newInstance();
            Field username = cls.getDeclaredField("username");
            username.setAccessible(true);
            username.set(object,"100");

            Field password = cls.getDeclaredField("password");
            password.setAccessible(true);
            password.set(object,"1234567");

            print(object.toString());
}

输出 
Test{username='100', password='1234567'}

反射方法

public static void main(String[] args){
    Class<?> cls = Class.forName("x.com.lib.Test");
            Object object = cls.newInstance();
            Method method = cls.getDeclaredMethod("showInfo",String.class,String.class);
        method.invoke(object,"hello","今天天气真好");
}


输出
name = hello
pwd = 今天天气真好

反射 获取方法返回类型

public static void main(String[] args){
        Class<?> cls = Class.forName("x.com.lib.Test");
                Object object = cls.newInstance();
                Method method = cls.getDeclaredMethod("showInfo",String.class,String.class);
                Class<?> returnType = method.getReturnType();
            int resulr = method.invoke(object,"hello","今天天气真好");
    }


输出     
name = hello
pwd = 今天天气真好
result = 2
returnType = int
注意

如果反射的方法是静态方法的话可直接invoke(null,...)

public static int showInfo(String name,String pwd){
      MyClass.print("name = " + name);
      MyClass.print("pwd = " + pwd);
      return 2;
  }

   public static void main(String[] args){
          Class<?> cls = Class.forName("x.com.lib.Test");
                  Object object = cls.newInstance();
                  Method method = cls.getDeclaredMethod("showInfo",String.class,String.class);
                  Class<?> returnType = method.getReturnType();
              int resulr = method.invoke(null,"hello","今天天气真好");
      }


  输出     
  name = hello
  pwd = 今天天气真好
  result = 2
  returnType = int

反射的如果是private属性的一定要注意兼容问题，因为如果别人哪天改了变量名,程序说不定crash了。
